# Psychology-First Development

Every game mechanic, scoring function, or agent behavior MUST be traceable to published research.

## Requirements
1. When adding a new scoring dimension, cite the paper and explain the operationalization.
2. When modifying agent behavior, check that it stays consistent with their Big Five profile and attachment style.
3. Never invent relationship dynamics without a theoretical basis. If unsure, flag it as speculative.
4. Parameter values (weights, thresholds) should be justified. The Gottman 5:1 ratio is 5:1 because the research says so, not because it "felt right."

## Anti-Patterns
- Making agents "nice" to produce happy outcomes. Let the psychology drive results.
- Cherry-picking theories that confirm desired outcomes. If anxious + avoidant = bad outcome, report it honestly.
- Using personality traits as decoration. Every trait must mechanically affect behavior or scoring.
